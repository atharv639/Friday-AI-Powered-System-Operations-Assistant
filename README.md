# **Friday – AI Powered System Operations Assistant**

### **Autonomous Desktop AI Agent with Voice Interface, Tool Calling, and Real-Time Hardware Telemetry**

**Friday is an advanced desktop AI assistant built in Python that combines Large Language Model reasoning, local neural voice synthesis, operating-system level automation, and real-time hardware telemetry into a single responsive interface.**

**Core Capabilities**
 - **AI Cognitive Core**
   - Gemini 2.5 Flash integration
   - Dynamic tool routing
   - Context-aware command execution
   - Function-calling architecture
 - **Operating System Control**
   - Launch applications
   - Manage processes
   - Install software via Winget
   - Uninstall software via Winget
   - Safe execution confirmation layer
 - **Voice Interface**
   - SpeechRecognition integration
   - Local XTTS_v2 voice synthesis
   - GPU accelerated inference
   - Asynchronous voice pipeline
 - **Real-Time Telemetry**
    - CPU monitoring
    - RAM monitoring
    - Disk monitoring
    - GPU VRAM monitoring
 - **Desktop HUD**
    - CustomTkinter interface
    - Real-time telemetry visualization
    - Animated dashboard
    - Dark-mode tactical UI
